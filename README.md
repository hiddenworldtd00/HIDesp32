# 🚀 HIDesp32 - Interface ESP32 Complète

## 📋 Description

HIDesp32 est une interface graphique complète pour ESP32 avec détection faciale, contrôle PC et toutes les fonctionnalités ESP32.

## ✨ Fonctionnalités

### 🔌 Connexion ESP32
- Détection automatique des ports COM
- Barre de progression 1-100%
- Connexion directe USB

### 📷 Vision par Ordinateur
- Détection faciale en temps réel
- Reconnaissance des expressions
- Suivi des yeux
- Détection de sourire
- Reconnaissance des gestes
- Contrôle par la tête

### 🎮 Contrôle PC
- Contrôle de la souris
- Contrôle du clavier
- Gestion du volume
- Capture d'écran
- Gestion des processus
- Informations système

### 🔧 Fonctionnalités ESP32
- GPIO (entrées/sorties)
- PWM (modulation de largeur d'impulsion)
- ADC (conversion analogique-numérique)
- DAC (conversion numérique-analogique)
- WiFi (mode STA/AP)
- Bluetooth (BLE)
- I2C, SPI, UART
- Deep Sleep
- Capteurs (DHT, BMP, etc.)
- Servo moteurs
- NeoPixel/LED
- Caméra ESP32-CAM
- OTA (mise à jour sans fil)

## 🚀 Installation

```bash
# Cloner le projet
cd HIDesp32

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
python run.py
```

## 📦 Dépendances

- Python 3.8+
- tkinter
- pyserial
- opencv-python
- mediapipe
- pyautogui
- psutil

## 🎯 Utilisation

1. Connectez votre ESP32 en USB
2. Lancez l'application
3. Attendez la détection automatique
4. Utilisez le menu pour accéder aux fonctionnalités

## 🖥️ Interface

- **Titre**: "HIDesp32"
- **Barre de progression**: 1-100% lors de la connexion
- **Menu**: Toutes les fonctionnalités ESP32
- **Onglets**: Organisation par catégories

## 🔒 Sécurité

- Confirmation avant les actions critiques
- Sauvegarde automatique des paramètres
- Gestion sécurisée des connexions
